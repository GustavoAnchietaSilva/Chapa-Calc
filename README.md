#  ChapaCalc

Sistema desktop desenvolvido em Python para automatizar o cálculo técnico de chapas de papelão utilizadas na fabricação de caixas maleta.

Projeto aplicado em ambiente industrial com foco em padronização de cálculos, redução de erros humanos e validação automática conforme limites operacionais de máquina.

---

##  Funcionalidades

✔ Cálculo automático da largura da chapa  
✔ Cálculo do comprimento com acréscimos industriais (aba, composição e refile)  
✔ Cálculo técnico de vinco  
✔ Validação automática conforme limites da máquina  
✔ Aprovação ou reprovação automática da chapa  
✔ Interface gráfica amigável  
✔ Geração de executável (.exe)  
✔ Instalador Windows via Inno Setup
